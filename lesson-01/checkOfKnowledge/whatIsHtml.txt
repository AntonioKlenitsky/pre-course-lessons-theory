
HTML (HyperText Markup Language) - это стандартизированный язык разметки, который используется для создания веб-страниц. 
Он определяет структуру содержимого веб-страницы с помощью различных элементов, таких как заголовки, параграфы, списки, изображения, ссылки и другие. 
HTML используется для описания того, как содержимое должно быть структурировано и отображено в браузере.

Тег div используется в HTML для создания блочного контейнера, который позволяет группировать другие элементы и применять к ним общие стили или свойства.

Тег input используется для создания различных формовых элементов, таких как текстовые поля, флажки, кнопки, переключатели и другие элементы ввода данных.

Тег select используется для создания выпадающего списка, который позволяет пользователю выбрать один из нескольких вариантов.

Тег img используется для вставки изображений на веб-страницу. Он указывает путь к изображению и может также содержать альтернативный текст для доступности.

Тег audio используется для вставки аудиофайлов на веб-страницу. Он позволяет воспроизводить звуковые файлы прямо на странице.

Атрибут тега - это дополнительная информация, которая указывается внутри открывающего тега и позволяет задавать дополнительные свойства или параметры элемента. 
Например, атрибут src у тега img указывает путь к изображению, а атрибут src у тега audio указывает путь к аудиофайлу. 
Таким образом, общий атрибут у тегов img и audio - это атрибут src, который используется для указания источника файла (изображения или аудио).

HTML (HyperText Markup Language) - это стандартизированный язык разметки, который используется для создания веб-страниц. 
Он определяет структуру содержимого веб-страницы с помощью различных элементов, таких как заголовки, параграфы, списки, изображения, ссылки и другие. 
HTML используется для описания того, как содержимое должно быть структурировано и отображено в браузере.

CSS (Cascading Style Sheets) - это язык таблиц стилей, который используется для оформления и визуального оформления веб-страниц. 
С помощью CSS можно определять цвета, шрифты, размеры, расположение элементов на странице, анимации и другие аспекты внешнего вида веб-сайта. 
CSS позволяет разделять содержимое и представление, обеспечивая гибкость и повторное использование стилей на различных страницах сайта.

Таким образом, HTML используется для создания структуры и содержимого веб-страницы, а CSS используется для управления ее визуальным оформлением. 
Оба языка часто используются вместе для создания красивых и функциональных веб-сайтов.

Тег style используется в HTML для определения стилей элементов на веб-странице. Он позволяет задавать CSS правила непосредственно внутри HTML документа.

Несколько примеров CSS свойств:

1. color: используется для задания цвета текста.
2. font-size: определяет размер шрифта.
3. background-color: устанавливает цвет фона элемента.
4. margin: устанавливает внешние отступы элемента.
5. padding: определяет внутренние отступы элемента.

Эти свойства позволяют управлять внешним видом элементов на веб-странице, такими как цвет текста, размер шрифта, цвет фона, отступы и многое другое.


Чем отличаются html/css/js?
Что такое тег script?


HTML, CSS и JavaScript - это три основных языка, используемых для создания веб-страниц и веб-приложений.
JavaScript - это язык программирования, который используется для добавления интерактивности на веб-страницы. 
Он может быть использован для создания динамического контента, обработки событий, анимаций, проверки данных и многого другого.

Тег <script> в HTML используется для встраивания JavaScript кода непосредственно в HTML документ. 
Это позволяет добавлять интерактивность и динамическое поведение на веб-странице. 
Тег <script> может быть использован для подключения внешних файлов с JavaScript кодом или для написания JavaScript кода прямо внутри HTML документа.


Какой атрибут нужно указать у тега script, чтобы подгрузить скрипт?
Для чего нам выносить код в отдельный файл\файлы?


Для подгрузки скрипта в HTML документ, вы можете использовать атрибут src у тега <script>. 
Этот атрибут указывает путь к внешнему файлу JavaScript, который вы хотите подключить. 
Например:
<script src="script.js"></script>

В этом примере script.js - это имя файла JavaScript, который вы хотите подключить.

Вынос кода JavaScript в отдельные файлы имеет несколько преимуществ:
1. **Модульность и организация**: Разделение кода на отдельные файлы позволяет лучше организовать проект, делая его более читаемым и легким для поддержки.
2. **Повторное использование**: Вынос кода в отдельные файлы позволяет повторно использовать его в различных частях вашего веб-сайта или веб-приложения.
3. **Кэширование**: Браузеры могут кэшировать внешние файлы JavaScript, что позволяет ускорить загрузку страниц и уменьшить использование ресурсов.
4. **Работа в команде**: При разработке в команде разделение кода на отдельные файлы делает процесс совместной работы более эффективным.

Таким образом, вынос кода JavaScript в отдельные файлы помогает улучшить организацию, повторное использование и обслуживание вашего кода.
